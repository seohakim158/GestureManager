# A trackpad gesture manager that works with AeroSpace

## Functionality
**3r** - move to the next non-empty aerospace workspace
**3l** - move to the previous non-empty aerospace workspace
**3u** - ignored (macos mission control)
**3ds** - quick workspace switching (aerospace workspace-back-and-forth)
**3dl** - show workspace switcher panel
**4ds** - quick app switching (simulates cmd-tab behaviour)
**4dl** - show app switcher panel
